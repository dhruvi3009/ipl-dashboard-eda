# 🏏 IPL Exploratory Data Analysis Dashboard (2007–2024)

Welcome to my IPL Analytics project! I am a 2nd-year Computer Engineering student, and this project began as an Exploratory Data Analysis (EDA) assignment using Python and Pandas. To make the insights accessible and engaging, I transformed my findings into this fully responsive, single-page web dashboard.

**🔗 [View the Live Dashboard Here](https://dhruvi3009.github.io/ipl-dashboard-eda/)** 

## 🛠️ Tech Stack & Tools Used
- **Data Engineering:** Python, Pandas (Jupyter Notebooks)
- **Data Source:** Kaggle (1,090 Matches, 260,920 Deliveries)
- **Frontend Architecture:** HTML5, Vanilla CSS (CSS Grid/Flexbox)
- **Data Visualization:** JavaScript, Chart.js

## 📊 Key Highlights & Features
Instead of a standard scrolling website, this project is built as a professional SaaS-style dashboard. It includes:
1. **All-Time Team Wins:** Highlighting the absolute dominance of MI and CSK.
2. **Toss Psychology:** Analyzing the gap between toss decisions (Field vs Bat) and actual match outcomes.
3. **Player Statistics:** Identifying the highest run-scorers, wicket-takers, and six-hitters in IPL history.
4. **Venue Analysis:** Breaking down the highest-scoring grounds and highlighting batting paradises like Visakhapatnam.
5. **Run-Scoring Trends:** A timeline showing the massive surge in average scores during the 2023 and 2024 seasons (Impact Player era).

## 💡 What I Learned
Through this project, I strengthened my ability to not only clean and manipulate massive datasets using **Pandas**, but also how to effectively communicate data insights to end-users using modern **Frontend Web Development** techniques without relying on heavy frameworks.
